# Go CRUD Server

A simple local Go HTTP server with basic GET, POST, PUT, DELETE routes.
Built for learning, testing, and open-source contributions.

## Features
- Lightweight Go HTTP server
- CRUD routes
- Local development friendly
- No database (in-memory storage)
- Easy to extend

## Getting Started

```bash
git clone https://github.com/yourusername/go-crud-server.git
cd go-crud-server
go run cmd/server/main.go
